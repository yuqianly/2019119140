
# 实验一 - GPIO 控制小熊派 LED1

实验一：根据小熊派 IoT 实验板电路（见资料目录），通过 STM32 CubeMX 构建控制 LED灯的工程，在 MDK下编译和仿真。要求实现以下三种显示模式：

    1、常亮；
    2、闪烁（比如：亮 1s 灭 1s）；
    3、呼吸灯模式（慢慢从灭到最亮，再从最亮慢慢回到灭的过程，1 分钟重复此过程 10次）。

## 补充说明

- 1.首先 LED(PC13) 先常亮2秒，然后进入闪烁模式和 PWM 呼吸灯模式

- 2.接线

    TIM16    LED(外接一个)
    PA6   -> 正极
    GND   -> 负极

- 3.注

    定时2 通过定时中断的方式让 LED(PC13) 灯闪烁。
    定时器16 用于产生 PWM。
